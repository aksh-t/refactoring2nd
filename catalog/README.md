# catalog配下の写経環境の整え方
1. ターミナルで `./catalog` まで移動する
2. `npm install` を実行する
  - `lodash` みたいに途中でパッケージを追加することもあるので、必要に応じて `npm install` する

# テストの実行の仕方
1. ターミナルで `./catalog` まで移動する
2. `npm run test [テストが書かれたJSファイルのパス]` を実行する
  - 例: `npm run test ./chap6/CombineFunctionsIntoTransform/examples/test.js`
